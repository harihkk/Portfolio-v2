This is a release summary derived from the DebugBrief changelog and PyPI release metadata.

DebugBrief is live on PyPI at version **1.3.0**, the latest of three public releases in the v1.x line (v1.1.0, v1.2.0, v1.3.0).

## Install

```bash
pipx install debugbrief
# or
uv tool install debugbrief
```

DebugBrief needs Python 3.9 or newer and uses native `git` when available. The project you debug does not need to be Python.

## What it does

Record the commands you run and the notes you write while debugging, then build an evidence-only Markdown report for a PR, a handoff, or an incident, derived strictly from what actually happened. No AI, no network, no telemetry.
